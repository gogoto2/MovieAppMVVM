<h1 align="center">MovieAppMVVM</h1>
<p align="center">
This app uses an API provided by https://www.themoviedb.org/. This project is based on movie app. In this project MVVM architecture, Alamofire, Codables, Generic Datasource, Core Data etc. are used.
</p>
